# Running Terraform

1. Initialize Terraform and download providers

```sh
terraform init
```

2. To see any changes that are required for the infrastructure

```sh
terraform plan
```

3. Format the code

```sh
terraform fmt
```

4. Validate the configuration files

```sh
terraform validate
```

5. Execute Terraform

```sh
terraform apply
```

6. Terminate Infrastructure

```sh
terraform destroy
```
